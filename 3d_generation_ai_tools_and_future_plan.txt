# 3D 생성/시뮬레이션 관련 AI 및 툴, 향후 개발 목록 및 목표

## 1. 3D 생성/시뮬레이션이 필요한 경우
- 현재 LLaVA/ChatGarment는 이미지-텍스트 대화(설명, 분석)만 지원하며, 직접적인 3D 변환/생성/시뮬레이션 기능은 내장되어 있지 않음.
- 2D 이미지를 3D로 변환하거나, 3D 의상/인체/포즈 시뮬레이션을 위해서는 별도의 3D 생성 AI/툴이 필요함.

## 2. 활용 가능한 3D 생성 AI/툴 예시
- **PIFuHD**: 2D 전신 이미지를 3D 인체 메쉬로 복원
- **DreamHuman**: 텍스트/이미지 기반 3D 인체 생성
- **Text2Mesh**: 텍스트 설명을 기반으로 3D 메쉬 스타일링
- **Meshy**: 텍스트/이미지에서 3D 모델 생성
- **ControlNet 3D**: 2D 이미지에서 3D 구조 추정
- **Stable Video Diffusion**: 2D 이미지에서 3D/동영상 변환
- **Blender, CLO3D, Marvelous Designer**: 3D 의상/인체 시뮬레이션 소프트웨어

## 3. LLaVA/ChatGarment와의 연계
- LLaVA/ChatGarment의 텍스트 설명 결과를 3D 생성 AI의 프롬프트로 활용 가능
- 예시 워크플로우:
  1. LLaVA로 이미지의 포즈/의상/외형을 텍스트로 추출
  2. 해당 텍스트를 3D 생성 AI(예: Meshy, PIFuHD 등)에 입력
  3. 3D 모델 생성 및 시뮬레이션

## 4. 향후 개발 목표
- LLaVA/ChatGarment의 텍스트 설명 결과를 자동으로 3D 생성 AI에 연동하는 파이프라인 개발
- 3D 의상/인체/포즈 시뮬레이션 자동화
- 사용자 맞춤형 3D 모델 생성 및 가상 피팅/시각화 지원
- 2D-3D 변환 AI와의 통합 및 워크플로우 자동화
